# 象棋助手-TheOne v1.1.0

第二个公开版本，发布于 2026-07-23。这次升级不是简单加深搜索时间，而是重做了“局面变化时，如何快速给答案并保证答案仍属于当前棋盘”的完整链路。

## 亮点

- Ultra 模式在约 2 秒发布初步结果，普通局面深化到约 6 秒，复杂或杀棋局面最多约 15 秒；
- 每次搜索绑定局面修订号，旧局面的迟到输出不能覆盖新局面；
- 增加离线开局库，但候选必须合法并通过 Pikafish 多候选复核；
- 分数接近时稳定显示，明显更优的着法或更短的杀棋仍会更新；
- 统一红方视角的分数与杀棋方向，补齐 `bestmove (none)` 终局处理；
- 增加墙钟超时、EOF/进程退出恢复、搜索停止与输出清空；
- 改进系统窗口过滤、目标窗口重绑定、多显示器框选坐标与反向棋盘规范化；
- 新增关键逻辑验收程序，覆盖窗口策略、局面方向、开局库、引擎终局、取消和搜索替换。

## 安装

1. 下载 `XiangqiAssistant-v1.1.0-macOS-arm64.zip`；
2. 解压并把 `象棋助手-TheOne.app` 拖入“应用程序”；
3. 首次打开若被 macOS 阻止，在 Finder 中右键应用并选择“打开”；
4. 在“系统设置 → 隐私与安全性 → 屏幕录制”中授予权限。

## 兼容性与限制

- 仅验证 Apple Silicon arm64 与 macOS 14+；
- 使用固定本地签名，尚未进行 Apple Developer ID 公证；
- 不保证适配所有棋盘皮肤、动画遮挡、缩放比例和变体；
- 公开构建排除鼠标控制模块，不会替用户自动落子；
- 搜索时间为预算，不是所有机器和局面的性能保证。

## 校验

```bash
shasum -a 256 -c XiangqiAssistant-v1.1.0-macOS-arm64.zip.sha256
```
