# 象棋助手-TheOne v1.0.0

首个公开版本，发布于 2026-07-22。

## 核心能力

- 通过 ScreenCaptureKit 选择并读取目标窗口；
- 使用 TheOne1006 ONNX 模型定位和识别 10×9 中国象棋棋盘；
- 生成 FEN，并通过本机 Pikafish UCI 引擎分析；
- 展示候选着法、评分、深度与主要变化；
- 使用连续帧与局面合法性检查保护识别结果；
- 以 macOS 菜单栏浮窗形式运行。

## 兼容性

- macOS 14+
- Apple Silicon arm64
- 固定本地签名，未使用 Apple Developer ID 公证
