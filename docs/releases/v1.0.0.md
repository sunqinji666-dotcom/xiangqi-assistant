# 象棋助手-TheOne v1.0.0

首个公开版本。它把“从屏幕看到棋盘”到“获得可复盘的候选着法”这一整条链路放在 Mac 本地完成。

## 亮点

- ScreenCaptureKit 读取用户明确选择的棋盘窗口。
- TheOne1006 ONNX 模型完成棋盘定位与 10×9 局面识别。
- Pikafish 在本机给出候选着法、评分、搜索深度和主要变化。
- 识别结果经过局面合法性与连续帧稳定性检查。
- 菜单栏常驻，使用 SwiftUI + AppKit 浮窗呈现分析结果。
- 当前发布 target 不包含自动点击或自动走棋能力。

## 安装

1. 下载 `XiangqiAssistant-v1.0.0-macOS-arm64.zip`。
2. 解压并把 `象棋助手-TheOne.app` 拖入“应用程序”。
3. 首次启动若 macOS 提示来源未验证，请在 Finder 中右键应用并选择“打开”。
4. 在系统设置中授予屏幕录制权限。

## 兼容范围

- macOS 14 或更高版本
- Apple Silicon（arm64）
- 当前未提供 Intel 版本

## 签名与公证

本版本使用固定本地签名，以维持稳定的应用身份和权限记录；尚未使用 Apple Developer ID 公证。因此其他 Mac 首次打开时可能需要手动确认。
