# 象棋助手-TheOne v1.3.0

发布说明：v1.3.0 将本机安装版中已经验证的最新能力整理为公开源码和可下载构建。

## 这次更新

- 千问先独立提出最多三套方案，随后由第二个低资源 Pikafish 在本机验棋。
- 千问第一轮全部不合格时独立重试一次；绿色推荐和引擎评分不会提前喂给千问。
- 新增紫色独立建议展示：风格、置信度、理由、计划、候选顺序和相对绿色的复算差距。
- 棋盘或行棋方变化后自动清除旧紫色箭头。
- 人工修正可通过唯一的一到两步真实变化迁移，不再固定在旧坐标。
- Pikafish 支持为第二验证引擎单独设置线程和 Hash。
- 千问视觉复核、千问建议都使用应用沙盒内的 `Application Support/象棋助手/ModelCredentials/qwen-dashscope` 凭证文件；公开源码不含凭证，也不含本机绝对路径。

## 安装

1. 下载 `XiangqiAssistant-v1.3.0-macOS-arm64.zip`。
2. 解压并把 `象棋助手-TheOne.app` 移入“应用程序”。
3. 在“系统设置 → 隐私与安全性 → 屏幕录制”中授予窗口读取权限。
4. 如需使用千问功能，将自己的 API Key 写入应用沙盒的 `Application Support/象棋助手/ModelCredentials/qwen-dashscope`。

## 限制

- 仅验证 Apple Silicon arm64 与 macOS 14+。
- 使用固定本地签名，尚未使用 Apple Developer ID 公证。
- 千问功能需要用户自行提供 API Key，并会在主动请求时向配置的兼容接口发送棋盘裁图或局面文本。
- Pikafish、ONNX Runtime、模型与其他第三方组件继续遵循各自许可证。

## 校验

```bash
shasum -a 256 -c XiangqiAssistant-v1.3.0-macOS-arm64.zip.sha256
```
