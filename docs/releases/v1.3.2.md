# 象棋助手-TheOne v1.3.2

这是基于当前本地源码重新构建的 Apple Silicon 发布版。

## 本次更新

- 同步当前棋盘识别、窗口校准、Pikafish 分析与千问建议代码。
- 千问建议会从本机筛选后的候选着法中选择，并补充简短计划。
- 凭证读取改为应用沙盒内的 Application Support 路径；公开源码和安装包不含本机绝对路径或 API Key。

## 安装

下载 `XiangqiAssistant-v1.3.2-macOS-arm64.zip`，解压后将 `象棋助手-TheOne.app` 拖入“应用程序”。首次打开如被 macOS 拦截，请在 Finder 中右键应用并选择“打开”。

当前构建使用固定本地签名，尚未 Apple Developer ID 公证。它只支持 Apple Silicon、macOS 14 及以上。

## 校验

```bash
shasum -a 256 -c XiangqiAssistant-v1.3.2-macOS-arm64.zip.sha256
```
