# 象棋助手-TheOne v1.2.0

发布于 2026-07-23。v1.2.0 集中解决真实使用中的连续识别问题：程序切换、多显示器、棋盘高亮、方向抖动、人工纠错和重新连接后，可信局面都应该继续存在。

## 亮点

- 每个象棋程序独立保存窗口相对棋盘区域，并保留旧格式恢复副本；
- 框选覆盖所有显示器，拖拽后必须点击“确认保存”；
- 使用最近五帧对 90 个交叉点分别投票，降低高亮和动画造成的闪烁；
- 首次可信识别后锁定本局方向，同时支持手动翻转；
- 支持按格修正棋子、恢复自动识别、手动同步行棋方；
- 支持只重建棋局跟踪状态，不丢失权限、窗口、框选和当前预览；
- 为不同捕获来源保存最近可信棋盘，窗口重建后可先恢复再后台刷新；
- 暂停后重新开始仍会强制执行全新扫描，不让旧局面成为新一轮证据；
- 引擎重试不再被显示成识别失败；
- 增加五帧逐格共识回归测试。

## 安装

1. 下载 `XiangqiAssistant-v1.2.0-macOS-arm64.zip`；
2. 解压并把 `象棋助手-TheOne.app` 拖入“应用程序”；
3. 首次打开若被 macOS 阻止，在 Finder 中右键应用并选择“打开”；
4. 在“系统设置 → 隐私与安全性 → 屏幕录制”中授予权限。

## 兼容性与限制

- 仅验证 Apple Silicon arm64 与 macOS 14+；
- 使用固定本地签名，尚未进行 Apple Developer ID 公证；
- 不保证适配所有棋盘皮肤、动画遮挡、缩放比例和变体；
- 公开构建排除鼠标控制模块，不会替用户自动落子；
- 五帧投票只改善完整盘面的稳定性，不会把缺失或非法局面伪装为有效局面。
